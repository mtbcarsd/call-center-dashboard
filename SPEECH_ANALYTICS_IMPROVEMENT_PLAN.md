# План улучшений речевой аналитики (на основе анализа встречи с конкурентом)

Источник: видеозапись `Встреча речевая аналитика 06.07.2026.mp4` — демо-встреча банка (МТ-банк) с вендором речевой аналитики. Транскрипция выполнена локально (faster-whisper, 74 минуты аудио).

## Что показал конкурент

- **Чек-листы**: кастомные весовые категории, LLM/copilot сам пишет промпт под категорию, поддержка морфологии слов, инверсия категорий («выполнили всё, кроме приветствия»)
- **Паузы/тишина в диалоге**: настраиваемый порог (например, 5 сек), льготный период для категорий вроде «оставайтесь на линии» — конкурент заявляет снижение показателя тишины на 25% после внедрения
- **Диаризация** (разделение оператор/клиент), в т.ч. для моно-записей (как у банка через Cisco)
- **Детекция негатива по позиции в звонке** (начало/середина/конец) — отдельная ML-модель, обученная на сотнях часов размеченных данных
- **AI-копайлот**: запрос на естественном языке → фильтр («покажи звонки Иванова за вчера»)
- **Гибкий конструктор графиков/дашбордов** с drill-down в конкретные звонки
- **Ручной override оценки** с пометкой «M» (manual) и историей изменений, тред комментариев супервайзер ↔ оператор
- **Геймификация**: личный рейтинг оператора среди коллег
- **Автоотчёты и рекомендации по оператору** через LLM
- **Тренировочный бот** (цифровой клиент + суфлёр) для обучения операторов
- Голосовые боты, обзвон, аудио-бейджи для очного обслуживания — не в тему колл-центра-дашборда, из плана исключены
